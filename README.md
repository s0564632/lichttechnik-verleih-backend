Lichttechnik-Verleih Backend

Dieses Repository enthält das Backend für die Semesteraufgabe **L**, eine Verwaltungs- und Verleihplattform für Lichttechnik-Equipment. Das Projekt wird fortlaufend über das Semester hinweg entwickelt.

## Aktueller Status
Das Grundgerüst des Express-Servers steht. Die Basis-Infrastruktur wurde erfolgreich eingerichtet und der Server nimmt Anfragen auf Port 3000 entgegen.

## Tech-Stack
* **Laufzeitumgebung:** Node.js (v24.x)
* **Paketmanager:** NPM (v11.x)
* **Framework:** Express.js (v4.x)
* **Versionsverwaltung:** Git & GitHub (inkl. GitHub Projects für Kanban-Management)
